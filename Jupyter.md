Note the Shortcut Key of Jupyter in VSCode.

# Order Mode

Pushing `Esc` to enter `Order Mode`, the following is the table of shortcut keys in `Order Mode`.

- A: Add `Code Cell` Above
- B: Add `Code Cell` Below
- M: Change `Code Cell` to `Markdown Cell`
- C: Change `Markdown Cell` to `Code Cell`
- D D: Delete the Cell


# Edit Mode

If you are in `Order Mode`, you could push `Enter` to enter `Edit Mode`.

- `Ctrl + Enter`: Run the current `Code Cell`
- `Shift + Enter`: Run the current `Code Cell` and Jump to the next `Code`
