> Q: How to read the .npy file in the path ./input/lab1.npy?


> A:

---

A .npy file is a binary file format used by NumPy to store arrays. To read ./input/lab1.npy, you can use numpy.load().

1️⃣ Basic Example
import numpy as np

data = np.load('./input/lab1.npy')

print(data)

This will load the array stored in the file into the variable data.

2️⃣ Check the Array Information (recommended)
import numpy as np

data = np.load('./input/lab1.npy')

print("Shape:", data.shape)
print("Data type:", data.dtype)
print("First few values:", data[:5])

This helps you understand the structure of the dataset.

3️⃣ If the .npy File Contains Pickled Objects

Sometimes .npy files store Python objects. In that case:

data = np.load('./input/lab1.npy', allow_pickle=True)

Only enable allow_pickle=True if you trust the file source.

4️⃣ Example in Jupyter Notebook
import numpy as np

data = np.load('./input/lab1.npy')
data

Jupyter will automatically display the array.

💡 Tip: If you want to convert the .npy file to something readable like CSV:

np.savetxt('lab1.csv', data, delimiter=',')

> Q: img.max 的输出是 <built-in method max of numpy.ndarray object at 0x7f1a6a8b80f0>，如何输出其具体数值？

> A:

---

你看到的输出：

<built-in method max of numpy.ndarray object at 0x7f1a6a8b80f0>

说明你打印的是方法本身，而不是调用它。在 NumPy 中，max 是一个方法，需要加 () 才会执行。

✅ 正确写法
img.max()

例如：

import numpy as np

img = np.load('./input/lab1.npy')

print(img.max())

输出会是类似：

255
📌 为什么会这样？

img.max → 方法对象（function reference）

img.max() → 调用方法并返回最大值

就像：

print(img.max)   # 方法
print(img.max()) # 方法执行结果
📊 另外两种等价写法

也可以使用 NumPy 的函数形式：

np.max(img)

或者指定维度：

img.max(axis=0)

💡 调试数组时常用的三件事：

print(img.shape)   # 数组维度
print(img.dtype)   # 数据类型
print(img.max())   # 最大值
print(img.min())   # 最小值

> Q: 已经有了 fig, (ax1, ax2) = plt.subplots(1, 2, figsize=(12, 5)) 如何给 ax1 和 ax2 分别加上 x 轴的标签？



> A:

---

在 Matplotlib 中，每个 Axes 对象（如 ax1、ax2）都有自己的 set_xlabel() 方法。你只需要分别设置即可。

✅ 示例
import matplotlib.pyplot as plt

fig, (ax1, ax2) = plt.subplots(1, 2, figsize=(12, 5))

ax1.set_xlabel("X axis of ax1")
ax2.set_xlabel("X axis of ax2")

plt.show()
📊 如果你同时画图（更常见）
import matplotlib.pyplot as plt
import numpy as np

x = np.arange(10)

fig, (ax1, ax2) = plt.subplots(1, 2, figsize=(12, 5))

ax1.plot(x, x)
ax1.set_xlabel("Input")

ax2.plot(x, x**2)
ax2.set_xlabel("Squared Input")

plt.show()
📌 相关常用设置

同样的方法还可以设置：

ax1.set_ylabel("Y label")
ax1.set_title("Plot 1")

ax2.set_ylabel("Y label")
ax2.set_title("Plot 2")

💡 小技巧（很多人不知道）：
如果你想给整个 figure 共用一个 x 轴标签：

fig.supxlabel("Shared X label")

> A:

---

你的代码报错主要是因为 axes[i].title(...) 和 axes[i].show() 不是 Axes 对象的方法。在 Matplotlib 中：

设置标题要用 set_title()

show() 是 plt.show()（属于 pyplot），不是 Axes 的方法

❌ 你代码中的错误
axes[i].title(f"Threshold: {threshold}")   # 错误
axes[i].show()                             # 错误

原因：

Axes 没有 title() 方法

Axes 也没有 show() 方法

✅ 正确写法
threshold_values = range(246, 256)

fig, axes = plt.subplots(1, len(threshold_values), figsize=(4*len(threshold_values), 4))

for i in range(len(threshold_values)):
    threshold = threshold_values[i]
    mask = generate_mask(img, threshold)

    axes[i].imshow(img, cmap='gray', vmin=0, vmax=255, alpha=0.7)
    axes[i].imshow(mask, cmap='Reds', alpha=0.5)

    axes[i].set_title(f"Threshold: {threshold}")

plt.show()
